# PKM Project

Repositori ini berisi folder-folder project PKM (Program Kreativitas Mahasiswa):

## Struktur Proyek

- **[CameraWebServer](file:///D:/PKM/CameraWebServer)**: Kode program server kamera berbasis ESP32.
- **[PROJECT ROBOT](file:///D:/PKM/PROJECT ROBOT)**: Halaman web monitoring/control berbasis Firebase.
- **[WEB_PKM](file:///D:/PKM/WEB_PKM)**: Kode Arduino/ESP32 untuk monitoring website.
- **[esp32-monitoring-web.ini](file:///D:/PKM/esp32-monitoring-web.ini)**: File konfigurasi.

## Cara Menggunakan

1. **WEB_PKM & CameraWebServer**:
   - Buka menggunakan Arduino IDE atau VS Code (dengan PlatformIO).
   - Pastikan board ESP32 sudah terinstal di Arduino IDE.
2. **PROJECT ROBOT**:
   - Berisi web monitoring Firebase.
   - Dapat di-deploy menggunakan Firebase CLI (`firebase deploy`).
